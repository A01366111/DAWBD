Definiciones

  • URL: es una dirección única que se le da a los archivos y páginas en internet para que puedan ser localizados.
  • MIME(Multipurpuse Internet Mail Extensions): se empleó originalmente para describir el contenido de los e-mails, pero luego se extendió para describir cualquier tipo de contenido, como ser la web.[1]
  • Métodos y Códigos de errores HTTP:
    • 1XX Respuestas informativas: este tipo de código reciba y da información sobre el proceso de la solicitud
    • 2XX Peticiones Correctas: en esta clase son de una operación exitosa
    • 3XX Redirecciones: "Aquellos códigos que comienzan con 3 indican que la solicitud ha sido recibida por el servidor. Sin embargo, para asegurar un procesamiento exitoso es necesario que el cliente tome una acción adicional. Este tipo de códigos aparecen principalmente cuando hay redirecciones."[4]
    • 4XX Errores del Cliente: En esta clase el cliente ha presentado un error, ya puede ser por solicitudes con errores, o están protegidos
    • 5XX Errores de Servidor: "Este tipo de respuestas indican que la solicitud correspondiente está temporalmente deshabilitada o es imposible de llevar a cabo. De nuevo, se generará automáticamente una página en formato HTML."[5]
  • XML (eXtensible Markup Language):Este lenguaje permite la organización para poder etiquetar documentos, y con esto puede definir lenguajes dependiendo de su función.
  • XHTML (Extensible Hypertext Markup Language): Lo que hace XHTML es unificar los lenguajes de HTML y XML, para poder hacer páginas HTML con element
  • HTML5: Es la más nueva versión de HTML agregando nuevas etiquetas y elementos para el desarrollo de páginsa web, agregando etiquetas para multimedia, efectos
  • JS: Es un lenguaje de programación que permite crear contenido dinámico en una página u apliación web.
  • PHP:Lenguaje adecuado para el desarrollo web, permite modificar dinámicamente el contenido de una página y permite establecer una conexión con la base
  • JSP: Es una tecnología basada en jav para que aplicaciones web se ejecuten en varios servidores.
  • Ajax: es una técnica que combina javascript con XML y "permite que una página web que ya ha sido cargada solicite nueva información al ser
