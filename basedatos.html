Cuestionario

  • ¿Qué es entonces una base de datos - DB?

    • Recolección de Datos que tienen que ver entre si, para generar información para tomar decisiones.

  • ¿En qué casos es conveniente usar bases de datos?

    • Cuánto se maneja muchos tipos de información en una empresa, como datos de empleados, ventas y gastos de una empresa para facilitar su lectura.

  • ¿Qué es un sistema de gestión de base de datos - DBMS?

    • "Es una colección de software muy específico, cuya función es servir de interfaz entre la base de datos, el usuario y las distintas aplicaciones utilizadas."[1]

  • Enlista y explica las funciones/responsabilidades que tienen los DBMS.

    • Facilidad de uso para que un usuario promedio pueda usarlo
    Colección de datos, Manipulación, Guardar, Producir Reportes
    Gran Volumen de entradas
    Gran Capacidad de Almacenamiento
    Poder almacenar a muchos Usuarios

  • Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

    • Fue en una empresa que se dedicaba a grabar eventos, lugares y construcciones, que teniamos que recolectar los datos de los clientes,
    los costos que ibamos a realizar para poder presentar ante ellos y justificar los costos del servicio, y haciamos los registros a mano.
